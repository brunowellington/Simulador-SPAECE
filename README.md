# Simulador do SPAECE - IDE MÉDIO

Sistema desenvolvido em React JS para auxiliar gestores escolares no cálculo de indicadores de rendimento educacional no SPAECE.

## Objetivo

A aplicação permite que o usuário informe:

- Pontuação de Língua Portuguesa (LP)
- Pontuação de Matemática (LM)
- IP - Fluxo

E obtenha automaticamente:

- LP Padronizado
- MT Padronizado
- IN - Desempenho
- IDE Médio

## 🛠️ Tecnologias Utilizadas

- React JS
- Vite
- Tailwind CSS
- React Hot Toast
